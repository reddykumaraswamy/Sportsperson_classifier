Sports Person Classifier
This is a demo webpage for our project: Image classification using Machine learning.
We made a machine learning model to classify and predict sports players by using their images. We currently used things like OpenCV, Haarcascades, etc to classify images with proper face and eyes visible. Our current model will be replaced with CNN and Deep Learning in our future work.
